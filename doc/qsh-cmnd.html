d テーブル名 アドレス
アドレス番地をテーブル名のデータとしてダンプする
<> ::= chan|ch|namec|dev|mount|mhead|mnt|pgrp|proc
<> d {up | ns} dump the current proc or namespace

Ex. <> d up show the current process table
<> d ns show the name space of the current process
